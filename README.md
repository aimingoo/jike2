这是极客时间课程《重学 TypeScript》的课件仓库，是我在极客时间里的第二门课程。



让我们一起在 TypeScript 的世界里，打造更快捷、更高效的未来！



### 简单说明

本仓库可以自由传播。

我会根据课程进度更新本仓库中的资源，在课程结束后，也会不定期的在这里上传一些新的材料和引用类资料。另外，我会关注所有 issues 中的问题，并尽可能及时地回复。
